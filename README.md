# Bonnie Ballet

Sitio web de la academia de ballet **Bonnie Ballet**, dirigida por Bonnie. Desarrollado como una página estática (HTML/CSS) con foco en presentar el estudio, las clases que ofrece y la comunidad.

## Estructura del proyecto

```
.
├── index.html          # Página principal del sitio
└── imagenes/           # Recursos visuales
    ├── hero/           # Imagen principal de portada
    ├── estudio/        # Fotos de clases y espacios del estudio
    ├── galeria/        # Galería de funciones e Instagram
    ├── otras-clases/   # Baile social y waltz para novios
    ├── directora-bonnie.jpg
    └── logo-negro.png
```

## Cómo verlo localmente

Solo abre `index.html` en cualquier navegador. No requiere servidor ni dependencias.

```bash
open index.html
```
